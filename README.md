# picoCTF Write-ups

Solutions et explications pour les challenges picoCTF que j'ai résolus.

## Organisation

Les write-ups sont classés par catégorie de challenge :

- **Binary Exploitation** - Exploitation de binaires et buffer overflows
- **Cryptography** - Challenges de crypto
- **Forensics** - Analyse de fichiers et récupération de données
- **General Skills** - Compétences de base
- **Reverse Engineering** - Désassemblage et analyse de code
- **Web Exploitation** - Vulnérabilités web

Chaque write-up détaille ma démarche, les outils utilisés et les concepts importants.

## Outils utilisés

- Python (pwntools, scripts custom)
- Ghidra / IDA pour le reverse engineering
- Wireshark pour l'analyse réseau
- CyberChef pour les manipulations rapides
- Burp Suite pour le web
- john / hashcat pour le cracking

## Avertissement

Ces solutions sont publiées dans un but éducatif. Je vous encourage à essayer de résoudre les challenges vous-même avant de regarder les solutions.
