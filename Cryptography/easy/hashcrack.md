# hashcrack

## Description 

A company stored a secret message on a server which got breached 
due to the admin using weakly hashed passwords. 
Can you gain access to the secret stored within the server?

```bash

Si on le passe dans CyberChef avec l'outil Analyse hash, on trouve
l'information suivante :

```

Je ne sais pas à quel point c'est utile.

En utilisant le site [Decode.fr](https://www.dcode.fr/hash-function) je trouve le
premier mot de passe.

la cmdline.
